# git-shortcuts
Som shortcuts that I use for GIT and BASH

Copy them in .bashrc:

*****************

alias l='ls -a'
alias gs='git status'
alias gcm='git commit -m'
alias gaa='git add .'
alias gpu='git push -u origin'
alias gp='git push origin'
alias gcob='git checkout -b'
alias gco='git checkout'
alias gcp='git cherry-pick -x'

*****************
